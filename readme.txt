TOP rádia z portálu PLAY.CZ - widget pro WordPress

== Description ==

Widget s TOP rádii z portálu PLAY.CZ. Počet rádií je nastavitelný v rozmezí 1 až 25. Data se aktualizují každých 5 minut.
Snadná úprava vzhledu přes CSS a třídy.

== Installation ==

1. Nahrajte `playcz-topradia` do adresáře `/wp-content/plugins/`
2. Aktivujte plugin v administraci WordPressu
3. V administraci (Vzhled -> Widgety) umístěte widget na vybranou pozici
